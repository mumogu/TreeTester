Spielanleitung
==============

* Wer mitspielen möchte schiebt seine Lösung in dieses Repo.
* Jeder Spieler muss eine mutable und eine immutable Lösung einreichen.
* Eure Lösungen müssen die Namen VornameMutableTree und VornameImmutableTree 
  tragen
* Eure Lösungen müssen von MutableTree bzw. ImmutableTree abgeleitet sein.
* Eure Lösungen müssen einen Binärbaum implementieren.

* Um Euch in Spiel einzutragen schreibt ihr einfach euren Vornamen in das Array 
  in Zeile 8 in TreeTester. 
* Der Tester möchte beim Aufruf drei Argumente haben. Das kann z.B. so aussehen
	java TreeTester results.csv 1000000 100000
  Die Ergebnisse werden im csv Format in result.csv geschrieben. Die kann man 
  einfach mit Excel öffnen und sich bunte Bilder malen lassen.
